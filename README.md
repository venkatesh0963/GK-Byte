# GK BYTE | Master General Knowledge.

![GK BYTE - Next-Generation EdTech Platform](https://img.shields.io/badge/Platform-EdTech-2563EB.svg)
![Status - Active](https://img.shields.io/badge/Status-Active-10b981.svg)

GK BYTE is a premier educational platform meticulously crafted to aid students, professionals, and competitive exam aspirants in mastering General Knowledge. Designed with a modern, dark-themed Bento UI, it provides a highly engaging and interactive learning experience.

## 🌟 Key Features

- **Comprehensive Coverage**: Highly curated resources covering 5+ core subjects including:
  - History
  - Politics
  - Economics
  - Current Affairs
  - Geography (and General Science)
- **Monthly Updates**: Fresh quiz modules and current affairs updates delivered every month to keep your knowledge up to date.
- **Subject Archives**: Browse through past monthly content and previous exam papers with ease.
- **Modern "Bento UI" Design**: A sleek, fully responsive dark theme interface that looks beautiful on desktops, tablets, and mobile devices.
- **Interactive Quizzes**: Test your knowledge with detailed explanations for correct and incorrect answers.

## 🛠️ Technology Stack

- **HTML5**: Semantic and accessible structure.
- **CSS3**: Custom CSS variables, modern grid/flexbox layouts, dark theme customization, and smooth animations (Glassmorphism & Claymorphism accents).
- **Vanilla JavaScript**: Lightweight interactions, theme toggling, scroll-to-top functionality, and dynamic pagination.
- **FontAwesome**: For scalable vector icons.
- **Google Fonts**: Inter (for body) and Poppins (for headings).
